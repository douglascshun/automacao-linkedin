## Ferramentas para proteção de estações de trabalho

**Introdução**

Neste módulo ferramentas para proteção de estações de trabalho, abordaremos algumas formas de proteger os arquivos de invasão e consequentemente roubo com o auxílio de algumas ferramentas disponíveis.

**Objetivos da aula**

Analisar as ferramentas e recursos disponíveis para proteção de estações de trabalho. Conhecer algumas aplicações de proteção de ransomware, phishing e proteção de malware, bem como o funcionamento de um firewall.

**Resumo**

**Ferramentas de proteção de estações de trabalho**

No  livro Os Cinco Anéis de Miyamoto Musashi consta uma frase interessante: “Não pense nos oponentes que não estão atacando; se preocupe com sua própria falta de preparação”.  Isso elucida bem o conceito que devemos considerar que não existem apenas ameaças externas, ou seja, ameaças que vêm somente do lado de fora do ambiente corporativo, mas sim, também podem existir ameaças internas, ou seja, de dentro da rede corporativa.

As estações de trabalho são computadores usados por profissionais em escritórios, empresas e organizações para realizar tarefas específicas relacionadas ao trabalho. Como essas estações de trabalho são frequentemente usadas para acessar informações confidenciais e importantes, é crucial protegê-las contra ameaças cibernéticas e garantir a segurança dos dados armazenados nelas.  
Existem diversas ferramentas de proteção de estações de trabalho disponíveis no mercado, cada uma com diferentes recursos e funcionalidades para ajudar a proteger o computador. Algumas das principais ferramentas de proteção de estações de trabalho incluem:  
- Antivírus: um software antivírus é uma ferramenta de proteção essencial que ajuda a detectar e remover malware, vírus e outras ameaças de segurança. Ele também pode bloquear sites maliciosos e downloads potencialmente perigosos para manter a estação de trabalho segura.  
- Firewall: um firewall é uma barreira de segurança que ajuda a proteger a estação de trabalho contra ameaças externas, bloqueando o tráfego de entrada não autorizado. Ele pode ser configurado para permitir apenas o tráfego de rede necessário para o trabalho e bloquear tudo além disso.  
- Anti-spam: o anti-spam é uma ferramenta de proteção que ajuda a bloquear e-mails indesejados, incluindo spam e phishing. Isso pode ajudar a reduzir o risco de ataques cibernéticos que visam obter informações confidenciais.  
- Gerenciamento de senhas: o gerenciamento de senhas é uma ferramenta que ajuda a garantir a segurança das senhas usadas para acessar sistemas e informações confidenciais. Ele pode gerar senhas seguras e armazená-las criptografadas em um cofre de senhas para evitar o acesso não autorizado.  
- VPN: uma VPN (Virtual Private Network) é uma rede privada que conecta a estação de trabalho a uma rede segura por meio da internet. Isso ajuda a proteger a estação de trabalho contra ameaças externas e permite que os usuários acessem informações confidenciais de forma segura de qualquer lugar.

Além dessas ferramentas, é importante que as estações de trabalho sejam atualizadas regularmente com as últimas atualizações de segurança e que os usuários adotem boas práticas de segurança, como não clicar em links suspeitos ou baixar software de fontes não confiáveis. Com as ferramentas de proteção adequadas e uma abordagem proativa para a segurança, é possível manter as estações de trabalho seguras e proteger os dados importantes armazenados nelas.

**Proteção contra ransomware**

Software anti-ransomware e seus benefícios

O ransomware é um tipo de malware que sequestra os dados do usuário e exige um resgate para recuperar o acesso a esses arquivos. Infelizmente, o ransomware é uma ameaça crescente e pode causar danos significativos, tanto para usuários domésticos quanto para empresas. Por isso, é crucial ter uma proteção eficaz contra ransomware para evitar ser vítima desse tipo de ataque.

Uma das formas mais eficazes de proteção contra ransomware é o uso de software anti-ransomware. Esse tipo de software é projetado especificamente para detectar e bloquear o ransomware antes que ele possa sequestrar os dados do usuário. Alguns dos benefícios do software anti-ransomware incluem:

- Detecção precoce: o software anti-ransomware é capaz de detectar o ransomware antes que ele possa criptografar os arquivos do usuário. Isso significa que o usuário pode tomar medidas para bloquear o ransomware antes que ele cause danos significativos.
- Bloqueio em tempo real: o software anti-ransomware pode bloquear o ransomware em tempo real, impedindo-o de sequestrar os arquivos do usuário. Isso significa que o usuário pode continuar a trabalhar normalmente sem se preocupar com a perda de dados.
- Prevenção de infecções futuras: o software anti-ransomware pode ajudar a prevenir futuras infecções de ransomware, bloqueando tentativas de acesso não autorizado aos dados do usuário.
- Fácil de usar: a maioria dos softwares anti-ransomware é fácil de usar e requer pouco ou nenhum conhecimento técnico. Isso significa que mesmo usuários com pouco conhecimento de tecnologia podem usá-lo com eficácia.
- Adaptação a novas ameaças: o software anti-ransomware é atualizado regularmente para se adaptar a novas ameaças de ransomware. Isso significa que o software é capaz de proteger o usuário contra as ameaças mais recentes.

No entanto, é importante lembrar que o software anti-ransomware não é uma solução completa de proteção contra ransomware. Os usuários também devem tomar outras medidas de segurança, como fazer backups regulares de seus dados e evitar abrir anexos de e-mails ou clicar em links suspeitos. Combinado com outras práticas de segurança, o software anti-ransomware pode ajudar a proteger os usuários contra o ransomware e manter seus dados seguros.

**Proteção contra phishing**

O phishing é uma técnica de engenharia social usada por criminosos para obter informações confidenciais dos usuários, como senhas, informações bancárias e detalhes de cartões de crédito. Os phishers geralmente usam e-mails falsos ou sites falsos para atrair as vítimas a fornecerem essas informações. A seguir, alguns exemplos de como se proteger contra o phishing:

- Verifique o remetente do e-mail: sempre verifique o endereço de e-mail do remetente antes de abrir um e-mail. Muitos e-mails de phishing usam endereços de e-mail falsos ou parecidos com endereços de empresas legítimas. Por exemplo, um e-mail falso do banco pode usar um endereço como "banco@banco123.com" em vez de "banco@banco.com".
- Não clique em links suspeitos: nunca clique em links em e-mails suspeitos ou em sites que você não reconhece. Esses links podem levá-lo a sites falsos que parecem legítimos, mas que na verdade são projetados para roubar suas informações. Em vez disso, digite o endereço do site diretamente na barra de endereço do seu navegador.
- Verifique a URL do site: verifique cuidadosamente a URL do site antes de inserir qualquer informação pessoal ou financeira. Muitos sites de phishing usam URLs falsas que são semelhantes aos sites legítimos, mas com erros de digitação ou nomes de domínio ligeiramente diferentes. Por exemplo, um site de phishing pode usar um URL como "www.banc0.com" em vez de "www.banco.com".
- Use software de segurança: use software de segurança, como antivírus e firewall, para proteger seu computador contra ataques de phishing. Esses programas podem ajudar a detectar e bloquear sites de phishing antes que eles possam roubar suas informações.
- Esteja ciente das táticas comuns de phishing: os phishers frequentemente usam táticas comuns para enganar as vítimas. Por exemplo, eles podem afirmar que sua conta foi comprometida e solicitar que você insira suas informações para "verificar" sua conta. Ou podem enviar e-mails de "urgência" que exigem que você aja imediatamente para evitar uma ameaça ou perda financeira. Esteja ciente dessas táticas e desconfie de qualquer e-mail que pareça urgente ou que solicite informações pessoais.

Em resumo, estar atento e tomar precauções de segurança pode ajudar a protegê-lo contra ataques de phishing. Nunca forneça informações pessoais ou financeiras a sites ou e-mails suspeitos e verifique cuidadosamente o remetente, o URL e as táticas usadas pelos phishers.

**Proteção contra malware**

Malware é a abreviação de "software malicioso" (em inglês, malicious software) e se refere a um tipo de programa de computador desenvolvido para infectar o computador de um usuário e prejudicá-lo de muitas formas. O malware pode infectar computadores e dispositivos de várias maneiras, além de assumir diversas formas, entre elas temos o vírus, worms, cavalos de Troia, spyware e outros. É importante que todos os usuários saibam como reconhecer e se proteger do malware.

**Como se proteger?** 

A pergunta mais importante de todas: "Como posso garantir que meu computador ou minha rede estão livres de malware?"  
A resposta tem duas partes: monitoramento pessoal e ferramentas de proteção. Uma das maneiras mais conhecidas de propagar malware é por e-mail, pois ele pode ser camuflado como uma mensagem de uma empresa conhecida, por exemplo, um banco, ou um e-mail pessoal de um amigo.  
Fique atento a e-mails que pedem suas senhas. Ou e-mails que parecem vir de amigos, mas que têm somente uma mensagem como "Você foi sorteado!", seguido de um link. O monitoramento  pessoal é o primeiro passo de proteção contra malware, mas apenas ter cuidado não é o bastante. Como a segurança corporativa não é perfeita, até os downloads feitos em sites legítimos podem conter um malware. Isso significa que até o usuário mais cuidadoso corre risco, a menos que você tome outras providências.

**Proteção contra malware**

O malware é um tipo de software malicioso que é projetado para danificar ou roubar informações do computador ou dispositivo do usuário. Aqui estão alguns exemplos de como se proteger contra o malware:

- Use software antivírus: use um programa antivírus confiável e mantenha-o atualizado regularmente. O software antivírus pode ajudar a detectar e remover malware antes que ele possa causar danos.
- Mantenha o software atualizado: mantenha o sistema operacional e o software do computador atualizados com as atualizações de segurança mais recentes. Os desenvolvedores de software frequentemente lançam atualizações para corrigir vulnerabilidades de segurança e impedir que o malware seja instalado.
- Não abra anexos de e-mail suspeitos: evite abrir anexos de e-mail de remetentes desconhecidos ou suspeitos. Muitos malwares são distribuídos por meio de anexos de e-mail ou links maliciosos.
- Evite baixar software de fontes não confiáveis: baixe software apenas de fontes confiáveis, como o site oficial do desenvolvedor ou uma loja de aplicativos confiável. Evite baixar software de sites de terceiros ou de fontes desconhecidas, pois eles podem conter malware.
- Use um firewall: um firewall pode ajudar a bloquear o acesso não autorizado ao seu computador ou rede. Certifique-se de configurar o firewall corretamente e mantê-lo atualizado.
- Esteja atento a sinais de infecção: fique atento a sinais de infecção por malware, como um computador mais lento ou comportamento estranho. Se você suspeitar que seu computador foi infectado, execute uma verificação completa do sistema com seu software antivírus.

Em resumo, a proteção contra malware requer a implementação de medidas de segurança proativas e o uso de software antivírus e firewall confiáveis. Mantenha o software e o sistema operacional atualizados e evite baixar software ou abrir anexos de e-mail de fontes não confiáveis. Se suspeitar que seu computador foi infectado, execute uma verificação completa do sistema com seu software antivírus.

**Firewall**

O termo firewall significa “barreira de fogo” e sua missão é impedir que dados, sistemas e ameaças externas vindas da rede WAN – no geral a internet – não acessem o ambiente dados, sistemas e dispositivos conectados à rede interna (LAN) da empresa, limitando o caminho das conexões, como as permissões de cada uma. Portanto, podemos dizer que um firewall é a barreira que permite ou não o fluxo de dados entre duas redes diferentes.

Para entender melhor o funcionamento de um firewall em uma infraestrutura de TI, imagine-o como sendo uma portaria de um prédio com um porteiro. Para que você tenha acesso aos apartamentos desse prédio, você precisa obedecer às regras estabelecidas pelo condômino do prédio, como se identificar, dizer em qual apartamento você vai, ser esperado pelo morador do apartamento, ser autorizado por esse morador para entrar no prédio e na maioria dos casos, não portar qualquer tipo de objeto que possa trazer risco à segurança do prédio; para sair, não se pode levar nada que pertence aos moradores ou ao prédio sem a devida autorização. Pois bem, o firewall é exatamente essa portaria e porteiro em nossa infraestrutura de TI, mas especificamente na rede interna da empresa.

Um firewall pode impedir uma série de ações maliciosas, como por exemplo, inibir que uma ameaça qualquer que tente se conectar a um computador dentro da rede LAN, sem a devida autorização, ou bloquear um determinado aplicativo que esteja enviando ou solicitando dados ou conexões não autorizadas. Firewall é um ponto entre duas ou mais redes no qual circula todo o tráfego. A partir deste tráfego é possível controlar e autenticar o tráfego, além de registrar por meio de logs, todo o tráfego da rede, facilitando sua auditoria. É um dos maiores destaques para hackers, pois se ele conseguir acessá-lo, pode alterar suas permissões e alcançar o bem mais valioso das empresas – a informação”. Podendo ser também definido como um componente ou conjunto de componentes que restringe o acesso entre uma rede protegida e a Internet, ou entre conjuntos de redes. (CHAPMAN, CHESWICK,STEVEBELLOVIN, 2016).

Os primeiros modelos de firewall foram inseridos em roteadores, no final da década de 80, por estarem em posição privilegiada, conectando redes distintas. As regras eram baseadas em ACLs de origem, destino e tipo de pacote. Porém, com o advento da WEB, foi necessário separar as funcionalidades do firewall dos roteadores. No geral, o uso mais comum do firewall é proteger uma rede privada, normalmente considerada segura e confiável, com dados e informações importantes, porém de acesso restritivo, contra acessos das inúmeras ameaças que podem tentar, sem autorização, acessar à rede privada para realizar ações mal-intencionadas, como por exemplo: o roubo de dados e informações, e a negação de serviços que possam comprometer as operações da empresa. Tecnicamente falando, um firewall é um dispositivo de hardware composto por duas ou mais interfaces de rede que será configurado de acordo com as políticas de segurança da informação e interligado às redes interna e externa de dados.

**Tópicos Avançados**

As ferramentas para proteção de estações de trabalho são essenciais para garantir a segurança dos sistemas e dados armazenados neles. Algumas das técnicas avançadas de proteção de estações de trabalho incluem:

Monitoramento contínuo: Ferramentas de monitoramento contínuo podem detectar e alertar sobre atividades suspeitas em tempo real, permitindo uma resposta mais rápida a incidentes de segurança.

Análise comportamental: Essa técnica usa algoritmos de inteligência artificial para analisar o comportamento do usuário e do sistema e identificar atividades anormais que possam indicar uma ameaça.

Sandboxing: Sandboxing é uma técnica que cria um ambiente isolado para executar aplicativos e arquivos suspeitos, permitindo que eles sejam testados sem prejudicar o sistema principal.

Controle de acesso: O controle de acesso é uma técnica que limita o acesso aos recursos do sistema com base nas permissões atribuídas a cada usuário. É importante garantir que apenas usuários autorizados possam acessar informações confidenciais.

Criptografia: A criptografia é uma técnica que usa algoritmos matemáticos para proteger dados confidenciais durante o armazenamento e transmissão. Isso ajuda a garantir que somente as pessoas autorizadas possam acessar esses dados.

Autenticação multifator: A autenticação multifator adiciona uma camada extra de segurança exigindo que os usuários forneçam mais de uma forma de identificação para acessar o sistema, como uma senha e uma impressão digital.

Virtualização: A virtualização é uma técnica que permite que vários sistemas operacionais e aplicativos sejam executados em um único dispositivo, o que ajuda a isolar o ambiente de trabalho e a proteger o sistema contra ameaças.

Ao implementar essas técnicas avançadas, as organizações podem melhorar significativamente a segurança de suas estações de trabalho, minimizando a probabilidade de ataques bem-sucedidos e garantindo a integridade e confidencialidade dos dados sensíveis.
