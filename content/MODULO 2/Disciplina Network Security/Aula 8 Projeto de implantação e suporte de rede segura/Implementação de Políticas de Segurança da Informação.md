# Definição
- É um documento formalizado por uma empresa, onde se lista conjunto de ações, regras, práticas, padrões, controles e técnicas específicas para a área da segurança das informações
- Define como os usuários devem manipular os dados com segurança
	 Um conjunto de regras = Política 

# Porque fazer uma política de Segurança da Informação?

Dado o ativo mais importante 

- A PSI visa assegurar os pilares da CIDAL das informações das empresas clientes, parceiros e funcionários, reduzindo riscos no ambiente de TI
- É uma estratégia que deve ser seguida por todos os usuários, com controles lógicos, físicos e humanos.

	Segurança da Informação não é só TI!

	Firewall - PSI = Problema!
