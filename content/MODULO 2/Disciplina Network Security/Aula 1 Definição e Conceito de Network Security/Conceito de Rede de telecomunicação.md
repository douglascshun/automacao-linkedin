[[Aula 1 Definição e Conceito de Network Security]]

Uma rede consiste em dois ou mais dispositivos, sejam computadores, celulares ou servidores conectados para compartilhar recursos, trocar informações, arquivos ou comunicações.
	Se comunicam por:
	- Cabos de rede
	- Ondas de Rádio
	- Feixes de Luz
	- Infra vermelho

# Tópicos importantes na arquitetura de redes

- Redes TCP/IP e WI-FI
- As sete camadas do modelo OSI
- NIC -Modem - Switch - Roteador - AP
- Topologia - Barramento - Pacotes
