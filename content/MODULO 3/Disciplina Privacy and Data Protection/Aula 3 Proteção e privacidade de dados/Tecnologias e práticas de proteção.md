[[Aula 3 Proteção e privacidade de dados]]

Práticas e tecnologias:

Descoberta de dados
	Descobrir quais dados estão sujeitos a regulamentação.

Prevenção contra perda de dados.
	Estratégias para evitar que dados sejam perdidos, roubados ou excluídos acidentalmente.

Backups
	Criar cópias de seus dados e armazena-los separadamente.

Snapshot
	Imagem completa de um sistema protegido, semelhante ao backup

Replicação
	Estratégias para copiar dados protegidos de um local para outro.

Criptografia
	Altera o conteúdo dos dados de acordo com um algoritmo.

Eliminação de dados
	Limita a responsabilidade ao excluir dados que não são mais necessários.

Autenticação e autorização
	Verificar as credenciais e garantir que os privilégios sejam aplicados corretamente.

Firewall
	Podemos utilizar firewall para que somente usuários autorizados trafeguem dados.
