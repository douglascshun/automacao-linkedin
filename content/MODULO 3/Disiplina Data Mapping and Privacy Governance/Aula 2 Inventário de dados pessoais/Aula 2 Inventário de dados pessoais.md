[[Data Mapping and Privacy Governance]]

**Introdução**

Caro(a) aluno(a), hoje você compreenderá sobre o inventário de dados pessoais. Inventário de dados pessoais é uma lista de todos os dados pessoais que uma organização detém, incluindo informações sobre indivíduos, como nomes, endereços, números de telefone, detalhes bancários e históricos médicos. Tudo isso a partir da absorção de dados que são transformados em informação. Legal, não?

**Objetivos da aula**

- Introdução e estudo de inventário pessoal;
- Identificar as diversas formas para administrar dados;
- Conhecer os diferentes tipos de dados, tipos de informações e conhecimentos.

**Resumo**

Vamos iniciar nosso resumo falando sobre inventário pessoal. Como foi dito anteriormente, é a lista de todos os bens pessoais de um indivíduo que são organizados para que eles possam ser facilmente localizados em várias circunstâncias (como é o caso de perda e roubo). Os bens podem ser imóveis, carros, roupas, móveis, livros, obras de arte, joias, dinheiro, contas bancárias, títulos, seguros e outros bens financeiros.

Um inventário pessoal também pode ser usado para criar um plano financeiro. Os indivíduos podem usá-lo para identificar os bens que possuem e os valores que estão associados a eles. Isso pode ajudar a calcular quanto dinheiro está disponível para investimentos, poupança e outras despesas. Além disso, poderá ajudar a identificar possíveis metas financeiras que uma pessoa possa ter.

  
**Existem vários tipos de inventários pessoais que podem ser criados:**

Inventário de bens: um inventário de bens lista todos os bens que uma pessoa possui, incluindo imóveis, carros, jóias, móveis, etc.

Inventário financeiro: um inventário financeiro lista todos os ativos financeiros de uma pessoa, incluindo contas bancárias, seguros, títulos e outros ativos.

Inventário de passivos: um inventário de passivos lista todas as dívidas de um indivíduo, como empréstimos, financiamentos, cartões de crédito e outras dívidas.

Inventário de impostos: um inventário de impostos lista todas as obrigações fiscais de uma pessoa, incluindo impostos federais, estaduais e locais.

Além dos tipos acima, existem outros tipos de inventários pessoais que podem ser úteis para os indivíduos. Por exemplo, alguém pode criar um inventário de investimentos para manter um registro de todas as suas ações, títulos, fundos mútuos e outras posições de investimento. Além disso, também é possível criar um inventário de planos de previdência para acompanhar todos os planos de previdência com os quais uma pessoa está envolvida. Os inventários também podem ser criados para acompanhar os gastos e a receita de uma pessoa.

Por outro lado, precisamos compreender a diferença entre dados, informação e conhecimento (Figura 2) que são pontos fundamentais na construção de um inventário pessoal.

Dados são coleções de fatos brutos, numéricos ou em forma de texto, que podem ser usados para um determinado propósito.

Informação é uma representação organizada de dados que é útil para um indivíduo ou organização. A informação é geralmente organizada, armazenada ou transmitida através de algum tipo de mídia.

Conhecimento é o entendimento de conceitos, procedimentos, habilidades e informações adquiridas por meio da experiência ou aprendizado. É a aplicação do conhecimento que leva até a criação de soluções para problemas reais.

**Vejamos alguns exemplos:**

Dados: Uma lista de endereços.

Informação: Um mapa mostrando os endereços em uma área específica.

Conhecimento: Uma compreensão das regras de trânsito para navegar a área de forma segura.

  
**Vejamos os diferentes tipos de dados:**

 - Dados numéricos: dados numéricos incluem números, porcentagens, medidas, datas e horas.

- Dados categóricos: os dados categóricos são valores de texto que descrevem uma característica.

- Dados geográficos: os dados geográficos são informações sobre a localização geográfica, como coordenadas, endereços, nomes de cidades e estados.

- Dados temporais: dados temporais são informações relacionadas ao tempo, como datas, horas e intervalos de tempo.

- Dados gráficos: os dados gráficos são informações visualizadas em gráficos, como gráficos de linha, barras e pizza.

  
**Os diferentes tipos de informações:**

- Texto: o texto é a forma mais comum de informação. Pode ser usado para fornecer detalhes sobre um tópico.

- Imagens: as imagens podem fornecer detalhes e informações visuais sobre um tópico.

- Vídeos: os vídeos são um tipo de informação visual que pode ser usado para fornecer informações detalhadas sobre um tópico.

- Áudio: o áudio é usado para fornecer informações em forma de áudio, como música ou discursos.

  
**Por fim, os diferentes tipos de conhecimentos:**

- Experiência: o conhecimento adquirido por meio da experiência é conhecimento prático.

- Aprendizado: o conhecimento adquirido por meio do aprendizado é conhecimento teórico.

- Intuição: a intuição é o conhecimento adquirido sem nenhum esforço consciente.

- Habilidades: as habilidades são conhecimentos adquiridos através da prática.

Portanto, compreendemos que dados, informação e conhecimento são termos relacionados, mas diferentes. Os dados são fatos brutos, a informação é uma representação organizada dos dados, e o conhecimento é o entendimento adquirido a partir dos dados e informações.
