[[Aula 3 Introdução ao mapeamento organizacional]]

Plano de ação:
	Cada organização tem um plano de ação diferente, e ele se baseia em diversos pontos como:
		- Tipos de dados pessoais
		- Volume de dados 
		- Etapas de fluxos de dados

Pontos essenciais:
	- Tecnologias
	- Locais de armazenamento
	- Origem dos dados
	- Compartilhamento de dados
	- Transferência internacional de dados (em casos)
	- Base legal
	- Políticas de privacidade
