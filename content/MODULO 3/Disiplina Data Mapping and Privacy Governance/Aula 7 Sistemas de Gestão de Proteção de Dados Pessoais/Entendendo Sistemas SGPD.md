[[Aula 7 Sistemas de Gestão de Proteção de Dados Pessoais]]

O que é a SGPD (Sistema de Gestão de Proteção de Dados Pessoais)?
	- Permite que a organização estabeleça uma governança eficaz e melhore continuamente a proteção de dados da organização.
	- Ao criar um SGPD, a proteção de dados torna-se parte integrante da governança corporativa.

Partes envolvidas:
	- Pessoas
	- Tecnologias 
	- Dados

Ponto Fundamental `Tratamento de dados`
